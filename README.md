This is an experimental bot, and contributions, bug reports, and code suggestions are welcome.

**Warning:** Do not use this bot on servers where automated clients are prohibited. Using it on unauthorized servers may result in a ban. Only use it in singleplayer worlds, on your own servers (ensure your hosting service allows bots on your server), or on servers that explicitly allow bots (such as some anarchy servers).

## Installation

1. Install Node.js if it is not already installed on your system.

2. Download the project ZIP file and extract it to a folder of your choice.

3. Open `minebot.mjs` and update the `host`, `port`, and `username` settings.

4. Open a terminal in the project folder.

5. Install dependencies:

   ```bash
   npm install
   ```

6. Start the bot:

   ```bash
   node minebot.mjs
   ```

7. Follow the sign-in instructions displayed in the terminal.

8. If the bot fails to join the server, restart it and try again.

9. Once connected, you can begin issuing commands to the bot in the terminal.
